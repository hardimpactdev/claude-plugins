# Changelog

All notable changes to the liftoff-laravel plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-05

### Added
- Initial release with 10 skills migrated from .ai/rules
- `laravel-general` - PHP 8.4 and Laravel 12 conventions
- `laravel-controllers` - Resourceful controller patterns
- `laravel-routes` - Waymaker attribute-based routing
- `laravel-validation` - FormRequest with DTOs
- `laravel-models` - Eloquent best practices
- `laravel-database` - Migrations, factories, seeders
- `laravel-dtos` - Laravel Data DTO patterns
- `laravel-enums` - PHP 8.4 backed enums
- `laravel-middleware` - Middleware patterns
- `laravel-testing` - Pest PHP v2 testing
- `/liftoff-laravel:quality-check` command
- Post-write PHP quality hooks
- Laravel Boost MCP integration
