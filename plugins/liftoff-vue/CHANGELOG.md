# Changelog

All notable changes to the liftoff-vue plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-05

### Added
- Initial release with 7 skills migrated from .ai/rules
- `vue-general` - Vue 3 Composition API patterns
- `vue-components` - liftoff-vue component library
- `vue-forms` - Inertia form handling with DTOs
- `vue-inertia` - Inertia.js v2 + Waymaker integration
- `vue-typescript` - TypeScript patterns (type over interface)
- `vue-tailwind` - Tailwind CSS v4 styling
- `vue-autoimports` - Auto-import configuration
