# Hello Command

A simple example command that demonstrates the slash command format.

## Usage

This command is invoked as `/template-plugin:hello` (namespaced by the plugin name).

## Instructions

When this command is invoked:

1. Greet the user warmly
2. Explain that this is a template command from the template-plugin
3. Provide a brief overview of what the plugin can do
4. Ask how you can help them today

## Example Response

"Hello! I'm the template plugin's greeting command. This plugin demonstrates how to build Claude Code plugins with commands, skills, agents, and hooks. How can I assist you today?"
