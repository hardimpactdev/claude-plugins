# Fix Command

Auto-fix PHP code issues.

## Usage

```
/php-quality:fix
```

## Instructions

Run auto-fix tools:

### 1. Rector

```bash
./vendor/bin/rector
```

Apply automated refactoring suggestions.

### 2. Pint

```bash
./vendor/bin/pint
```

Fix code style issues.

### 3. Report Changes

List all files that were modified.
