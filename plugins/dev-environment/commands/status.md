# Environment Status

Check the status of the local development environment.

## Usage

```
/dev-environment:status
```

## Instructions

Run the following commands and report the results:

```bash
# Check Launchpad status
launchpad status --json

# List all sites
launchpad sites --json
```

Provide a summary including:
- Whether services are running
- Number of sites configured
- Any services that are not running
- PHP versions in use
